# SmartCampus API

## API Endpoints

- **Discovery:** `GET /api/v1`
- **Rooms:**
    - `GET /api/v1/rooms`: List all rooms.
    - `POST /api/v1/rooms`: Create a new room.
    - `GET /api/v1/rooms/{roomId}`: Get room details.
- **Sensors:**
    - `GET /api/v1/sensors`: List all sensors.
